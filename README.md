# Web Company Backend NodeJS (Typescript) + Prisma

## How To

```shell
pnpm dev install
```

then :

```shell
pnpm firstrun
```

## Endpoints

### Testimony

```
/testimonies
/testimonies/{id}
```

### Hero

```
/hero/right
/hero/left
/hero/{id}
```

### Partner

```

```

### Services

# Web Company Backend NodeJS (Typescript) + Prisma

## How To

```console
pnpm dev install
```

then :

```console
pnpm firstrun
```

or manually
`pnpm install` => `pnpm prisma db push` => `ts-node dbseeder.ts`

## TO run after command above

```console
pnpm dev
```
